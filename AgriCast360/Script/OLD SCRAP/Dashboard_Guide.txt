Create these measures in Power BI by going to **Modeling â†’ New Measure**

---

#### Basic Aggregations

```DAX
// Total Records
Total Records = COUNT(Price_Data[Modal_Price])

// Average Price
Average Price = AVERAGE(Price_Data[Modal_Price])

// Total Commodities
Total Commodities = DISTINCTCOUNT(Price_Data[Commodity])

// Total Markets
Total Markets = DISTINCTCOUNT(Price_Data[Market])

// Total Trading Value (approximate)

```

---

#### Price Analysis Measures

```DAX
// Minimum Price
Min Price = MIN(Price_Data[Min_Price])

// Maximum Price
Max Price = MAX(Price_Data[Max_Price])

// Price Range
Price Range = [Max Price] - [Min Price]

// Average Volatility
Average Volatility = AVERAGE(Price_Data[Price_Volatility_%])

// Weighted Average Price (by volume, if available)
Weighted Avg Price = 
DIVIDE(
    SUMX(Price_Data, Price_Data[Modal_Price] * 1),
    COUNT(Price_Data[Modal_Price])
)
```

---

#### Time Intelligence Measures

```DAX
// Previous Month Price
Previous Month Price = 
CALCULATE(
    [Average Price],
    DATEADD(Date_Table[Date], -1, MONTH)
)

// Month-over-Month Change
MoM Change = [Average Price] - [Previous Month Price]

// MoM % Change
MoM % Change = 
DIVIDE(
    [MoM Change],
    [Previous Month Price],
    0
)

// Year-to-Date Average
YTD Average Price = 
CALCULATE(
    [Average Price],
    DATESYTD(Date_Table[Date])
)

// Moving Average 7-Day
MA7 = 
CALCULATE(
    [Average Price],
    DATESINPERIOD(Date_Table[Date], LASTDATE(Date_Table[Date]), -7, DAY)
)

// Moving Average 30-Day
MA30 = 
CALCULATE(
    [Average Price],
    DATESINPERIOD(Date_Table[Date], LASTDATE(Date_Table[Date]), -30, DAY)
)
```

---

#### Comparative Measures

```DAX
// Price vs Market Average
Price vs Market Avg = 
VAR MarketAvg = CALCULATE([Average Price], ALL(Price_Data[Market]))
VAR CommodityPrice = [Average Price]
RETURN CommodityPrice - MarketAvg

// Price Index (Base = 100 in January)
Price Index = 
VAR BasePrice = 
    CALCULATE(
        [Average Price],
        Date_Table[Month] = 1
    )
RETURN DIVIDE([Average Price], BasePrice, 0) * 100

// Rank by Price
Price Rank = 
RANKX(
    ALL(Price_Data[Commodity]),
    [Average Price],
    ,
    DESC,
    DENSE
)

// Top N Commodities
Is Top 10 Commodity = IF([Price Rank] <= 10, TRUE(), FALSE())
```

---

#### Statistical Measures

```DAX
// Standard Deviation
Price Std Dev = STDEV.P(Price_Data[Modal_Price])

// Coefficient of Variation (CV)
Price CV = DIVIDE([Price Std Dev], [Average Price], 0) * 100

// Price Percentiles
Price 25th Percentile = PERCENTILE.INC(Price_Data[Modal_Price], 0.25)
Price 50th Percentile = PERCENTILE.INC(Price_Data[Modal_Price], 0.50)
Price 75th Percentile = PERCENTILE.INC(Price_Data[Modal_Price], 0.75)

// Interquartile Range
IQR = [Price 75th Percentile] - [Price 25th Percentile]
```

```DAX
RETURN DIVIDE(MonthlyAvg, AnnualAvg, 1) * 100

// Volatility Score (normalized 0-100)
Volatility Score = 
VAR MaxVol = CALCULATE(MAX(Price_Data[Price_Volatility_%]), ALL())
VAR CurrentVol = [Average Volatility]
RETURN DIVIDE(CurrentVol, MaxVol, 0) * 100

// Trend Direction
Trend Direction = 
VAR CurrentPeriod = [Average Price]
VAR PreviousPeriod = [Previous Month Price]
RETURN
    SWITCH(
        TRUE(),
        CurrentPeriod > PreviousPeriod * 1.05, "Strong Up",
        CurrentPeriod > PreviousPeriod, "Up",
        CurrentPeriod < PreviousPeriod * 0.95, "Strong Down",
        CurrentPeriod < PreviousPeriod, "Down",
        "Stable"
    )
```

**Step 10: Add Core Measures**

Copy and paste each measure into _Measures table:

```dax
// 1. Total Records
Total Records = COUNTROWS(Price_Data)

// 2. Average Modal Price
Avg Modal Price = AVERAGE(Price_Data[Modal_Price])

// 3. Price Range
Price Range = MAX(Price_Data[Max_Price]) - MIN(Price_Data[Min_Price])

// 4. YoY Growth
YoY Price Growth = 
VAR CurrentYear = MAX(Price_Data[Year])
VAR PrevYear = CurrentYear - 1
VAR CurrentPrice = CALCULATE([Avg Modal Price], Price_Data[Year] = CurrentYear)
VAR PrevPrice = CALCULATE([Avg Modal Price], Price_Data[Year] = PrevYear)
RETURN
DIVIDE(CurrentPrice - PrevPrice, PrevPrice, 0)

// 5. Market Share
Market Share % = 
DIVIDE(
    COUNTROWS(Price_Data),
    CALCULATE(COUNTROWS(Price_Data), ALL(Price_Data[Market]))
) * 100

// 6. Price Volatility Avg
Avg Volatility = AVERAGE(Price_Data[Price_Volatility_%])

// 7. Top Commodity
Top Commodity = 
CALCULATE(
    MAX(Price_Data[Commodity]),
    TOPN(1, ALL(Price_Data), [Avg Modal Price], DESC)
)

// 8. Weather Impact Score
Weather Impact = 
AVERAGE(Weather_Data[temperature_2m_mean]) * 0.4 +
AVERAGE(Weather_Data[precipitation_sum]) * 0.3 +
AVERAGE(Weather_Data[rain_sum]) * 0.3
```

**Step 11: Add Comparison Measures**

```dax
// Month-over-Month
MoM Change = 
VAR CurrentMonth = MAX(Price_Data[Month])
VAR PrevMonth = CurrentMonth - 1
VAR CurrentPrice = CALCULATE([Avg Modal Price], Price_Data[Month] = CurrentMonth)
VAR PrevPrice = CALCULATE([Avg Modal Price], Price_Data[Month] = PrevMonth)
RETURN
DIVIDE(CurrentPrice - PrevPrice, PrevPrice, 0)

// MoM Change % Formatted
MoM % = 
FORMAT([MoM Change], "0.00%")

// Previous Month Price
Prev Month Price = 
CALCULATE(
    [Avg Modal Price],
    DATEADD(DateTable[Date], -1, MONTH)
)
```
