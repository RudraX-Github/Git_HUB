### Insight #1: Use Ensemble Methods
- Price-based features have very high correlation (>0.95)
- Linear models may already achieve RÂ² > 0.80
- **Recommendation**: Use XGBoost/LightGBM for non-linear patterns
- **Expected**: RÂ² 0.85-0.92

### Insight #2: Don't Shuffle Time Series Data
- Dataset has temporal order (Arrival_Date)
- **Must use**: Time-series split, not random split
- **Must avoid**: Predicting past from future
- **Method**: Train on first 80%, test on last 20%

### Insight #3: Weather Features Still Matter
- Weather features show 0.41 average correlation
- **Why low?**: Normalization reduces feature variance
- **Solution**: Tree-based models capture non-linear relationships better
- **Expected**: XGBoost/LightGBM will use these effectively

### Insight #4: Commodity Variation Exists
- Different commodities may have different patterns
- **Options**: 
  1. Single unified model (simpler)
  2. Commodity-specific models (more complex, higher accuracy)
- **Try first**: Unified model; upgrade if needed
